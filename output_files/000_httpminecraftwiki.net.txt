<div class="mcwiki-header" style="margin: 3px; font-weight: bold; text-align: center; font-size: 120%;"> About Minecraft</div>
<div style="padding: 0em 1em;">
<p><b><a href="/wiki/Minecraft" title="Minecraft">Minecraft</a></b> is a <a href="https://en.wikipedia.org/wiki/Open_world" class="extiw" title="wikipedia:Open world">sandbox</a> construction game, inspired by <a href="/wiki/Infiniminer" title="Infiniminer">Infiniminer</a>, <a href="https://en.wikipedia.org/wiki/Dwarf_Fortress" class="extiw" title="wikipedia:Dwarf Fortress">Dwarf Fortress</a> and <a href="https://en.wikipedia.org/wiki/Dungeon_Keeper" class="extiw" title="wikipedia:Dungeon Keeper">Dungeon Keeper</a>, created by <a href="/wiki/Notch" title="Notch">Markus Persson</a>, the founder of <a href="/wiki/Mojang_AB" title="Mojang AB">Mojang AB</a>. The game involves players creating and destroying various types of blocks in a <a href="/wiki/The_Overworld" title="The Overworld">three dimensional environment</a>. The player takes an avatar that can destroy or create <a href="/wiki/Blocks" title="Blocks">blocks</a>, forming fantastic structures, creations and artwork across the various <a href="/wiki/Multiplayer" title="Multiplayer">multiplayer</a> servers in multiple <a href="/wiki/Gameplay" title="Gameplay">game modes</a>.
</p><p>Minecraft is available to players for €19.95 (~£17.21/~$26.90). Singleplayer and multiplayer, when purchased, can be played in the user's web browser, or using the downloadable client. Minecraft <a href="/wiki/Classic" title="Classic">Classic</a> is available to play <span class="plainlinks"><a rel="nofollow" class="external text" href="http://www.minecraft.net/classic/play">for free</a></span>. Minecraft development started around the 10<sup>th</sup> of May 2009, pre-orders for the full game started being accepted on the 13<sup>th</sup> of June 2009. Minecraft's official release date was the 18<sup>th</sup> of November 2011.
</p>
